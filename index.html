D2Coding 폰트 다운로드 페이지

🔗 **기반 GitHub 저장소:** baehk31/D2coding

아래 링크를 클릭하면 해당 폰트 파일이 **바로 다운로드됩니다.**

1. 설치형 폰트 (.ttf / .otf)

PC에 설치하여 사용하는 파일 형식입니다.

  • D2Coding.ttf (일반) 다운로드
  • D2CodingBold.ttf (굵은 글씨) 다운로드
  • D2Coding.otf (일반) 다운로드

2. 웹폰트 (.woff / .woff2)

웹사이트 개발 시 `@font-face` 규칙에 사용하는 파일 형식입니다.

  • webfont/D2coding.woff2 (일반, 최신 표준) 다운로드
  • webfont/D2coding.woff (일반, 범용 표준) 다운로드
  • webfont/D2codingBold.woff2 (굵은 글씨) 다운로드
